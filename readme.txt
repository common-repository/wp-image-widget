== Description ==

Image Widget - New Version : A simpe and easy way to place an image in Your widget area.

1. Download plugin zip file to your computer and extract in a folder
1. Upload `wp-image-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
